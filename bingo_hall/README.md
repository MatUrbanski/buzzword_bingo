# BingoHall

## Installation

1. Install Elixir dependencies:

    ```sh
    mix deps.get
    ```

2. Install Node.js dependencies:

    ```sh
    cd assets && npm install
    ```

3. Make sure all the tests pass:

    ```sh
    mix test
    ```

4. Fire up the Phoenix endpoint:

    ```sh
    mix phx.server
    ```

5. Visit [`localhost:4000`](http://localhost:4000) to play the game!
